## 引用
　　使用[CocoaPods](http://cocoapods.org)可以很方便地引入AYDispatcher。Podfile添加AYDispatcher的依赖。

```ruby
pod "AYDispatcher"
```

## 简介
　　AYDispatcher用于解决在多模块开发时，接口调度的解决方案。详细使用方案可以查看Example示例。
